# 📈 PEAD Earnings Agent

A powerful automated agent that tracks **Post-Earnings Announcement Drift (PEAD)** for Indian stock markets (BSE/NSE). This tool scrapes official earnings announcements, parses key financial data, and alerts users about high-momentum stocks.

## 🚀 Features

- Automated scraping of earnings announcements from BSE & NSE
- Intelligent data parsing and structuring
- Persistent storage with database integration
- Smart logic to identify high-drift stocks
- Real-time alerts for actionable opportunities
